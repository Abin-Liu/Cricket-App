工作原理

魔兽大蛐是一个能够自动与魔兽世界《熊猫人之谜》中5.2版本以后新增的首领战斗宠物反复战斗，通过战胜NPC使玩家角色获得大量经验（依据角色等级而定，大致等于交一个同级任务所获得经验值的75%左右），从而实现无人值守式练级的外挂程序，其工作原理如下：

  1. 自动使用预先选定的宠物进行组队，并设置出最优化的宠物技能与组合。
  2. 与NPC交互并开始宠物战斗。
  3. 在战斗过程中自动选择最适合当前战况的技能以达到理想的战术目的。
  4. 战斗胜利后玩家角色获得经验值，并自动返回步骤1。

系统需求

  • Microsoft Windows 2000或更高版本的32位Windows系列操作系统。
  • 魔兽世界游戏。

安装与运行

  1. 关闭魔兽世界游戏。
  2. 运行下载的魔兽大蛐安装程序。
  3. 启动魔兽大蛐，软件会在系统托盘区添加一个图标。如下图所示：

  4. 运行魔兽世界，进入“按键设置”界面。
  5. 在“按键设置”界面找到“魔兽大蛐”，为“主动作控制”进行按键绑定。如下图所示：

  6. 在“按键设置”界面找到“选择目标按键”，记住“与目标互动”功能使用的按键，如果尚未绑定，请绑定一个按键。如下图所示：

  7. 返回Windows桌面，右键点击托盘区的魔兽大蛐图标，在弹出菜单进行选“按键设置”，在各个项目中填入与游戏中相一致的按键。如下图所示：

  8. 进入游戏后在屏幕中间找到“魔兽大蛐”的标题按钮，你可以用鼠标把它拖动到任何地方。点击这个按钮将打开插件主界面，如下图所示：

  9. 点击“魔兽大蛐”插件界面的“幼宠”标签，将你希望升级的低级宠物拖到插件中，幼宠等级必须在6级以上，且必须拥有至少400血量以防被秒杀。

  10. 点击“魔兽大蛐”插件界面的“战宠1”标签，将你所拥有的所有满级的赞达拉撕踝者（巨兽之岛，可交易）和赞达拉袭胫者（巨兽之岛，可交易），或虚空精灵龙（菲拉斯）和精龙宝宝（菲拉斯稀有掉落，可交易）拖到插件中，强烈推荐赞达拉撕踝者与赞达拉袭胫者，这两种宠物可以将战斗效率提升3倍以上。

  11. 点击“魔兽大蛐”插件界面的“战宠2”标签，将你所拥有的所有满级的剥石者幼崽（地狱火半岛）和昆莱小雪人（昆莱山）拖到插件中，如果你没有这两种宠物，则可暂时使用熊类（黑海岸幼崽或海加尔幼熊），熊类对小动物无免伤，故不建议使用。

  12. 前往四风谷的汇风岭，靠近野外首领宠物“幸运的小艺”，确定你所处的位置与NPC足够近（可以直接用鼠标右键交谈），如下图所示：。

  13. 在游戏中按下“Pause”键，程序开始运行。

额外功能

  1. 定时关机


    2. 本程序可以设置定时关闭游戏及计算机（需要系统管理员权限）的计划任务，设置方法为在Windows桌面右下角的“魔兽大蛐”图标上点击右键，在弹出菜单中选择“自动关闭”并设置关闭游戏或计算机的时间。如下图所示：

  2. 自动排副本


    3. 本程序提供自动排进副本功能，使DPS职业在动辄好几十分钟的排本时间中可以一边刷小艺一边离开电脑去做自己的事情，当排到时程序会帮你进入副本并发出闹铃声提醒你回到电脑前进行游戏。使用方法：在挂小艺的过程中以正常方式排副本即可，注意在排本过程中确保程序处于运行状态，否则角色可能因为静止时间过长而掉线。

常见问题及解决方法

  1. 人物站在那里发呆，不进行任何动作。


    2. 答：请正确设置各个快捷键，并确认角色已站在NPC近旁，并检查是否已将各相应宠物拖入插件界面，以及这些宠物是否处于等待复活状态。

  2. 我连一只像样的满级宝宝都没有，就算现在去抓几只，要把它们练到25也太累了，有什么好建议吗？


    3. 答：找基友借几只25级的赞达拉撕踝者和赞达拉袭胫者（最好能借到6只，一步到位），再借1只25级的海加尔幼熊（这几种宠物都可以交易）。自己去黑海岸抓3只满攻的黑海岸幼崽作为幼宠挂到25级，然后告诉基友你借的那几只宝宝被你不小心卖掉了/摧毁了/盗号了/BUG消失了/下雨淋湿融化了/煮方便面时烧死了，如果实在过意不去，就把海加尔幼熊归还。现在你已经拥有了一支战斗力一流的强力战宠队伍，快乐地开始挂吧。

  3. 怎么能快捷地让1级幼宠拥有足够的等级和400血量，确保不会被秒杀呢？


    4. 答：先在暴风城找1级松鼠或兔子打一场，幼宠会升到2级，然后去四风谷找野外的泽地小歌手打一场，即可升到5-6级，基本不可能被秒杀了。

  4. 我的复活宠物CD完全不够用，大部分时间在等待复活中。


    5. 答：你所拥有的满级战宠数量不够，去多抓/买/借几只吧！

  5. 如果我不想练幼宠，那么宠物栏的第三格该放什么宠物？


    6. 答：如果你没有设置幼宠，则插件只操作第1、2个宠物栏位，对第3栏不作任何干预，只有当前2只战宠全部战死后，插件才会让第3只宠物出战并只使用1号技能。理论上这种事根本不会发生，所以你爱放啥放啥。

  6. 我经常在与首领宠物对战中败北。


    7. 答：正常情况下2只战宠并肩作战可以轻而易举地击败NPC，胜率绝对在95%以上。如果你的战宠不是灰色品质的话，那么也许你该去整容了。
